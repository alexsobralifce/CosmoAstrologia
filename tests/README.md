# Testes

Esta pasta contém todos os arquivos de teste do projeto.

## Arquivos de Teste

- **test_astrology_calc.py** - Testes de cálculos astrológicos
- **test_groq_integration.py** - Testes de integração com Groq
- **test_rag.py** - Testes do sistema RAG
- **test_registro.py** - Testes de registro de usuários
- **test_update.py** - Testes de atualização
- **test_update_flow.py** - Testes de fluxo de atualização
- **test_user_data.py** - Testes de dados de usuário
- **test_frontend_update.md** - Documentação de testes do frontend
- **find_aquarius_asc.py** - Script para encontrar Ascendente em Aquário

## Como Executar

Execute os testes Python diretamente:

```bash
python tests/test_astrology_calc.py
python tests/test_registro.py
# etc...
```
