# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [Unreleased]

### Adicionado
- Sistema de deploy automatizado para GitHub → Vercel (frontend) e Railway (backend)
- Documentação completa de deploy em DEPLOY.md
- Arquivos .env.example para configuração de variáveis de ambiente
- GitHub Actions workflow para verificação de build
- Melhorias na página de Trânsitos Astrológicos com linha do tempo horizontal
- Exemplos práticos em todas as descrições de trânsitos
- Personalização única da seção "Como isso impacta sua rotina" para cada trânsito

### Melhorado
- Sistema de cores padronizado baseado no Figma
- Layout responsivo usando Flexbox e CSS Grid
- Formatação didática das interpretações astrológicas
- RAG (Retrieval-Augmented Generation) com busca expandida
- Integração com Groq para interpretações mais ricas

### Corrigido
- Erros de sintaxe no código
- Problemas de CORS em produção
- Configuração de variáveis de ambiente
- Build do frontend para produção

## [1.0.0] - 2024

### Adicionado
- Sistema completo de autenticação (email/senha e Google OAuth)
- Dashboard interativo com múltiplas seções
- Cálculo de mapa astral completo
- Interpretações astrológicas com IA
- Sistema de trânsitos futuros
- Sinastria (compatibilidade entre signos)
- Biorritmos
- Sistema de temas claro/escuro
- Internacionalização (PT/EN)
